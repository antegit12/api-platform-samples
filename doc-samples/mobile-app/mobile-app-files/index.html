If you don't see any movies below, it probably means that you did not change the URL in /www/index.html to point to your UserGrid app.

You can make the change in XCode. In the file index.html, look for the function

$.ajax({ url: "https://api.usergrid.com/{your_org}/{your_app}/movies"

After you do so, be sure to stop and start the simulator to see your changes.
